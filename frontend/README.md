# GoFood Frontend

Frontend React application for the GoFood food delivery platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Environment Variables

Create `.env` file:
```env
REACT_APP_API_URL=https://your-backend-url.com
```
